# rails-interview / MR
Minimalistic checklists app with realtime updates through websockets + API

## Requeriments

 - Ruby with bundler
 - Yarn
 - Chrome (only for selenium tests)

## Setup

1. `bundle install`
2. `rails db:setup`
3. `yarn install`

## Run server

`$ ./bin/dev`

## Run tests

`$ bundle exec rspec`
